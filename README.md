### Installation

1. Clone the repository:
```bash
git clone https://github.com/jmvilla12/portfolio-jose.git

```

2. Install dependencies:
```bash
pnpm install
```

3. Start the development server:
```bash
pnpm run dev
```

4. Open your browser and navigate to `http://localhost:5173`
